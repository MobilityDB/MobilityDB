Detailed Description

Bounding box operators for temporal network points.

These operators test the bounding boxes of temporal network points, which are STBox boxes. The following operators are defined: overlaps, contains, contained, same The operators consider as many dimensions as they are shared in both arguments: only the space dimension, only the time dimension, or both the space and the time dimensions.
