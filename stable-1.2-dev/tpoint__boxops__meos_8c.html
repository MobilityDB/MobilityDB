Detailed Description

Bounding box operators for temporal points.

These operators test the bounding boxes of temporal points, which are an STBox, where the x, y, and optional z coordinates are for the space (value) dimension and the t coordinate is for the time dimension. The following operators are defined: overlaps, contains, contained, same, and adjacent.

The operators consider as many dimensions as they are shared in both arguments: only the space dimension, only the time dimension, or both the space and the time dimensions.
