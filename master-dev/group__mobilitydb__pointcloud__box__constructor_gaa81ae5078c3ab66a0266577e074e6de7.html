Datum Pcpoint_to_tpcbox ( PG_FUNCTION_ARGS  )

Return a degenerate (single-point) TPCBox built from a pcpoint.

Spatial bounds equal the point's X/Y/[Z]. SRID comes from the schema; pcid from the pcpoint itself. The per-backend schema cache makes this possible without dragging pgpointcloud's PG-layer API into MEOS. @sqlfn tpcbox()
