Sample a Raquet raster chip along a tgeompoint trajectory.

The chip is identified by its QUADBIN cell, which encodes the Web-Mercator tile coordinates and thus the full georeferencing without any separate metadata. Instants outside the tile extent or on nodata pixels are silently dropped; NULL is returned when no instants survive.

Parameters
[in]pixelsRow-major pixel bytes (all bands interleaved or single-band depending on the Raquet producer)
[in]widthTile width in pixels (typically 256)
[in]heightTile height in pixels (typically 256)
[in]quadbinCARTO QUADBIN cell identifier (uint64)
[in]pixtypePixel data type
[in]nodataNodata sentinel value
[in]has_nodataWhether nodata filtering is active
[in]trajInput tgeompoint trajectory (SRID 4326)
Returns
tfloat instant set, or NULL
