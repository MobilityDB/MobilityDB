Detailed Description

PG V1 wrappers for the lifted temporal tquadbin cell operations.

The shared DGGS operations (resolution, validity, parent, centroid point, boundary, area) route through the generic tcellindex_* entry points, which dispatch to the quadbin_cellops descriptor. The quadbin-unique quadkey lift calls the typed tquadbin_cell_to_quadkey directly because the quadkey string has no DGGS analogue and is not part of the descriptor.
