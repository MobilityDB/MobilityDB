Detailed Description

Bounding box for pgpointcloud temporal types.

Mirrors STBox (spatiotemporal box) but carries an additional pcid field — pgpointcloud schema id — because TPCBoxes from different schemas cannot meaningfully be compared / unioned (the underlying dimensions are schema-specific). Fixed-size struct; no varlena.

Flag bits live in MEOS_FLAGS_* (see meos_internal.h): X (bounds present), Z (z-dimension present), T (time span present), GEODETIC (geographic coords). A TPCBox must have at least one of X or T.
