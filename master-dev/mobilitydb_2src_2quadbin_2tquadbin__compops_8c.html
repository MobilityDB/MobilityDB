Detailed Description

PG V1 wrappers for the tquadbin comparison operators.

Each wrapper unpacks a Quadbin from a bigint Datum (the SQL surface for bare cells) or a Temporal from a TOAST datum, calls the tquadbin-specific MEOS function declared in meos_quadbin.h, and returns the result.
