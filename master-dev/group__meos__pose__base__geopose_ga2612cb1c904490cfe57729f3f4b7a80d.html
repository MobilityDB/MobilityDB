Return the OGC GeoPose JSON representation of a pose.

Parameters
[in]posePose value
[in]conformanceConformance class to emit (Basic-Quaternion or Basic-YPR)
[in]precisionDecimal places to keep in the JSON numbers; pass a negative value to use json-c's default
Returns
On error return NULL

The Basic conformance classes mandate a geographic outer frame. Poses with SRID 0 are treated as geographic; poses with non-zero non-4326 SRIDs are rejected (use WKT/WKB instead). @csqlfn Pose_as_geopose()
