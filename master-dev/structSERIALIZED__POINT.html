Detailed Description

Serialized varlena layout for a single pcpoint.

Mirrors SERIALIZED_POINT in pgsql/pc_pgsql.h. The on-disk byte layout is load-bearing in every existing pgPointCloud database; the upstream PR moves this typedef into lib/pc_api.h without changing the layout.
