Detailed Description

Clipper2-backed polygon Boolean engine for MEOS.

Replaces the bespoke Martinez-Rueda port with vendored Clipper2 v2.0.1. Public C entry point used by the _mdb_internal_clip_* SQL functions (and ultimately by the temporal-point clipping path in tgeo_spatialfuncs.c).
