Pose * pose_from_geopose ( const char *  json)

Return a pose from its OGC GeoPose JSON representation.

Parameters
[in]jsonGeoPose JSON string (Basic-YPR or Basic-Quaternion)
Returns
On error return NULL

Auto-detects the conformance class from the JSON keys present:

  • quaternion: Basic-Quaternion (returns a 3D pose).
  • angles: Basic-YPR (returns a 3D pose; or 2D if pitch=roll=0 and h=0, since that is the canonical 2D representation).

Position is parsed as {lat, lon, h} (degrees, degrees, metres). The resulting pose has SRID 4326 (WGS-84). Projected SRIDs are not supported by this entry point — use the WKT/WKB I/O for those. @csqlfn Pose_from_geopose()
