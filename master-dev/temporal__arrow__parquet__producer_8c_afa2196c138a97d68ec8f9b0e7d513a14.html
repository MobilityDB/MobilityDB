int producer_write_decomposed ( const char *  path)

Write the per-row decomposed value-leaf ground truth to path.

One line per value, in population order:

row_id|kind|subtype|interp|srid|t0:LEAF0,t1:LEAF1,...[#REFHEX]

kind is the base type tag, subtype/interp are MEOS's own temporal_subtype / temporal_interp strings, srid is the SRID the Arrow top-level slot carries (tspatial_srid for the SRID-bearing types, 0 otherwise), t is the integer microsecond timestamp, and LEAF is the value-leaf payload exactly as the kernel decomposes it, computed from the original Temporal* with MEOS's own per-type accessors. For a rigid geometry the constant reference-geometry EWKB hex is appended once after a #. A zero-MEOS process reads the same fields back from Parquet and compares them value-exact.

Returns
1 on success, 0 on failure
