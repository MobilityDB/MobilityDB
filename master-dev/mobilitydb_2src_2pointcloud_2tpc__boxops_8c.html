Detailed Description

PG wrappers for TPCBox-based bounding-box operators on the pgPointCloud temporal types.

Mirrors the stbox / tspatial bbox surface in mobilitydb/src/geo/tgeo_boxops.c.

The five families exposed are overlaps (&&), contains (@>), contained (<@), same (~=), and adjacent (-|-). Each is wired to:

  • tpcbox vs (tpcpoint | tpcpatch) via Boxop_tpcbox_tpointcloud
  • (tpcpoint | tpcpatch) vs tpcbox via Boxop_tpointcloud_tpcbox
  • tpointcloud vs tpointcloud via Boxop_tpointcloud_tpointcloud

tstzspan-based variants (tpcpoint vs tstzspan, etc.) are not registered here — the generic Boxop_temporal_tstzspan in mobilitydb/src/temporal/temporal_boxops.c already handles every temporal type, so the SQL operator declarations in 420_tpcpoint.in.sql / 430_tpcpatch.in.sql can target it directly.
