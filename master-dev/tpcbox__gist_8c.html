Detailed Description

R-tree GiST index for the TPCBox bounding-box type.

Mirrors mobilitydb/src/geo/tspatial_gist.c — same five support functions (consistent / union / penalty / picksplit / same), specialized to TPCBox. picksplit reuses the generic bbox_gist_picksplit helper, which is layout-aware of T_TPCBOX (binary-compatible STBox prefix).
