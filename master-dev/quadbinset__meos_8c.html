Detailed Description

MEOS implementations of the set-returning quadbin functions, returning quadbinset.

Shape of every function below:

  1. Ask the quadbin kernel for the cell array (palloc'd).
  2. Pack the array into a Datum array.
  3. set_make_free the Datum array — the constructor copies into its own storage and frees the input.
  4. Return the Set.
