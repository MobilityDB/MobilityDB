Detailed Description

PG wrappers for the TPCBox bounding-box type.

Fixed-size struct (no varlena), so recv / send simply shuttle the bytes through. in / out delegate to MEOS-layer tpcbox_in / tpcbox_out, which read and write the WKT text form.
