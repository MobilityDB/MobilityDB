Detailed Description

A simple program that demonstrates the RTree index for searching MEOS bounding boxes: floatspan, tstzspan, tbox, and stbox.

The program runs all four bounding box types, inserting random boxes into the index and verifying search results against a brute-force scan.

The program can be built as follows

gcc -Wall -g -I/usr/local/include -o rtree_example rtree_example.c -L/usr/local/lib -lmeos -lproj
