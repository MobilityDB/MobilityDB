Detailed Description

PG wrappers for the tpcpoint lifted temporal type.

Most SQL bindings delegate to the generic Temporal_* PG wrappers in mobilitydb/src/temporal/temporal.c — the generic parser / output / constructor / accessor path dispatches through MeosType and works correctly for T_TPCPOINT because the base type is wired into basetype_in_state / basetype_out / datum_cmp / datum_eq / datum_hash. This file holds only the per-type accessors that need to know they are looking at pcpoint data — namely pcid and the schema-aware dimension projections.
