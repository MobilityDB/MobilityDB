static bool derive_quadbin ( GDALDatasetH  ds,
uint64 *  quadbin,
const char *  label 
)
static

Derive the QUADBIN cell of a raster tile from an open GDAL dataset's EPSG:3857 geotransform and spatial reference.

The dataset must carry an EPSG:3857 (Web-Mercator) spatial reference and an axis-aligned (north-up) geotransform describing a single QUADBIN tile; a missing/rotated geotransform or a non-Web-Mercator reference is an error.

Returns
true on success; on failure sets a MEOS error and returns false
