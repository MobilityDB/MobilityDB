Detailed Description

Internal declarations for the tquadbin type-inheritance boilerplate.

This header is the analogue of meos/include/h3/th3index.h. It carries the extern decls for the static helpers in tquadbin.c that do not belong in the public meos_quadbin.h. The validity macro VALIDATE_TQUADBIN(temp, ret), used in every lifted function and public accessor, lives in the public meos_quadbin.h alongside the other type-validation macros so bindings can use it directly.

Quadbin is the square/Web-Mercator (planar) counterpart of the hexagonal/geodetic H3 index, so its temporal point bridge uses tgeompoint rather than tgeogpoint.
