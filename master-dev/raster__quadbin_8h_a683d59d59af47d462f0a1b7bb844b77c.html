uint64 * trajectory_quadbins ( const Temporal *  traj,
uint32_t  zoom,
int *  count 
)

Return the unique QUADBIN cells at zoom covered by a trajectory.

Suitable for use as the WHERE-clause argument when joining against a Raquet table:

SELECT raster_tile_value_quadbin(band_data, 256, 256, quadbin, ...)
FROM elevation_raquet
WHERE quadbin = ANY(trajectory_quadbins(traj, 8));
Parameters
[in]trajInput tgeompoint trajectory (SRID 4326)
[in]zoomRaquet zoom level (0–15)
[out]countNumber of distinct cells returned
Returns
Palloc'd array of QUADBIN cell identifiers
