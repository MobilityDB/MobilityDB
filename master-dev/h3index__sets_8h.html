Detailed Description

Set-returning static helpers on h3index.

These are the MobilityDB ports of the nine h3-pg SETOF-returning functions. Each call returns a finite collection of H3 cells (or icosahedron face indexes), packaged as the right MobilityDB set type — h3indexset for cell collections, intset for the face index list.

All functions here operate on STATIC h3index values — they are not lifted to the temporal th3index type. The temporal lift (producing a tset<h3indexset> or similar) is parked pending a tset<T> primitive design.

libh3 is the ultimate authority for semantics; this file is a thin allocate / call / wrap layer:

size-query → palloc → libh3 fill → filter zeros → set_make_free.
