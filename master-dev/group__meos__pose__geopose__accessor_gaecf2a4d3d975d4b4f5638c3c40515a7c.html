Temporal * tpose_apply_geo ( const Temporal *  temp,
const GSERIALIZED *  body 
)

Return the world-frame trajectory obtained by applying a temporal pose to a body-frame geometry.

Per instant the rigid-body transform R(q_i) · v + p_i is applied to the body geometry; the result is a temporal point with the same subtype and interpolation as the input tpose. v1 supports point and multipoint body geometries — the output is a tgeompoint carrying the per-instant transformed point. Linear interpolation of the resulting trajectory is the chord on each segment, which approximates the true rigid-body trajectory (a circular arc under SLERP); the same linear-interpolation trade-off MobilityDB already uses for spatial trajectories.
