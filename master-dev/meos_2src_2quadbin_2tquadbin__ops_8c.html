Detailed Description

Datum-convention wrappers for the quadbin static kernel and the quadbin_cellops descriptor that plugs quadbin into the shared temporal cell-index machinery (meos/src/temporal/tcellindex.c).

A quadbin cell is a uint64 carried in a Datum with the int8/bigint convention (Int64GetDatum / DatumGetInt64). The cell centroid is emitted as a planar tgeompoint in the cell's lon/lat (SRID 4326); a Web-Mercator (SRID 3857) emission would set point_srid = 3857 and reproject here.
