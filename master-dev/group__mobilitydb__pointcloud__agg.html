Detailed Description

Aggregate functions for the pgpointcloud temporal types (extent, tcount, wcount, merge, appendInstant, appendSequence).

Note
pcpointset / pcpatchset have no type-specific PG wrappers: SQL bindings dispatch directly to the generic Set_* PG wrappers in mobilitydb/src/temporal/set.c. Same-pcid uniformity is enforced inside set_make_exp via ensure_valid_pcpointset_pcpoint / ensure_valid_pcpatchset_pcpatch from the MEOS layer.
