Detailed Description

PG wrappers for per-type pcpoint / pcpatch accessors.

Set-level SQL bindings for pcpointset / pcpatchset delegate to the generic Set_* wrappers in mobilitydb/src/temporal/set.c (dispatch happens via the Oid-MeosType cache). This file hosts the type-specific accessors: the trivial pcid reads and the schema-aware dimension getters (getX, getY, getZ, getDim) that build on top of the PCSCHEMA cache.
