Temporal * tpose_from_geopose ( const char *  json)

Return a temporal pose from its OGC GeoPose JSON representation.

Parses the TemporalGeoPose envelope (see tpose_as_geopose for the shape). Each instant's payload is auto-detected as Basic-Quaternion or Basic-YPR by the same rule as the static entry point. The interpolation and bounds-inclusion flags come from the envelope; the resulting tpose has SRID 4326.
