double pose_angular_distance ( const Pose *  pose1,
const Pose *  pose2 
)

Return the shortest-arc angular distance (radians) between two poses' orientations.

For 2D poses this is the shortest-arc difference in theta, accounting for the (-π, π] wrap. For 3D poses this is the SLERP arc angle 2 · acos(|q1 · q2|) under the quaternion double-cover convention (the absolute value gives the shortest of the two equivalent representations). The dot product is clamped to [-1, 1] to absorb the small numeric drift |q| - 1 that long compositions can introduce.

Parameters
[in]pose1,pose2Poses (must agree on dimension)
Returns
On error return DBL_MAX
