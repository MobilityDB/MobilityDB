static uint8_t * tbox_to_wkb_buf ( const TBox *  box,
uint8_t *  buf,
uint8_t  variant 
)
static

Write into the buffer a temporal box in the Well-Known Binary (WKB) representation.

The output is as follows

  • Endian byte
  • Flag byte stating whether the are a value and/or a time dimensions
  • Output the 2 doubles for the value dimension (if there is one) and the 2 timestamps for the time dimension (if there is one)
