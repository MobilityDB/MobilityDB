A simple program that reads from a CSV file synthetic trip data in Brussels generated by the MobilityDB-BerlinMOD generator https://github.com/MobilityDB/MobilityDB-BerlinMOD splits the trips and the speed of these trips by tiles, and computes aggregate values of the count, duration, and distance of the tiles.

Please read the assumptions made about the input file in the file 05_berlinmod_disassemble.c in the same directory. Furthermore, the program that we already have computed the extent of the trips and the speeds. Otherwise, a first scan of the CSV file is needed for computing both extents and then proceed for the second scan as below.

The program can be build as follows

gcc -Wall -g -I/usr/local/include -o 07_berlinmod_tile 07_berlinmod_tile.c -L/usr/local/lib -lmeos
