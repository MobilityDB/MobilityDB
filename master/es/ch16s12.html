Comparaciones

Comparaciones Siempre y Alguna Vez

El operando de prueba puede ser un identificador de celda H3 h3 simple o otra trayectoria th3; ambos son soportados en cualquier posición de operando.

  • Devolver true si una celda H3 temporal es alguna vez igual a la prueba en al menos un instante

    ever_eq({h3,th3},th3) → boolean

    ever_eq(th3,h3) → boolean

    {h3,th3} ?= th3

    th3 ?= h3

  • Devolver true si una celda H3 temporal es siempre igual a la prueba a lo largo de todo su eje de tiempo

    always_eq({h3,th3},th3) → boolean

    always_eq(th3,h3) → boolean

    {h3,th3} %= th3

    th3 %= h3

  • Devolver true si una celda H3 temporal es alguna vez diferente de la prueba

    ever_ne({h3,th3},th3) → boolean

    ever_ne(th3,h3) → boolean

  • Devolver true si una celda H3 temporal es siempre diferente de la prueba

    always_ne({h3,th3},th3) → boolean

    always_ne(th3,h3) → boolean

Comparaciones Temporales

  • Devolver un booleano temporal que da la igualdad (o desigualdad) por instante entre una celda H3 temporal y una prueba

    temporal_teq({h3,th3},th3) → tbool

    temporal_teq(th3,h3) → tbool

    temporal_tne({h3,th3},th3) → tbool

    temporal_tne(th3,h3) → tbool
