Operations for Temporal Types: Analytics Operations

Simplification

  • minDistSimplify, minTimeDeltaSimplify: Return a temporal float or a temporal point simplified ensuring that consecutive values are at least a certain distance or time interval apart

  • maxDistSimplify, douglasPeuckerSimplify: Return a temporal float or a temporal point simplified using the Douglas-Peucker algorithm

Reduction

  • tsample: Sample a temporal value with respect to an interval

  • tprecision: Reduce the temporal precision of a temporal value with respect to an interval computing the time-weighted average/centroid in each time bin

Similarity

  • hausdorffDistance: Return the discrete Hausdorff distance between two temporal values

  • frechetDistance: Return the discrete Fréchet distance between two temporal values

  • frechetDistancePath: Return the correspondence pairs between two temporal values with respect to the discrete Fréchet distance

  • dynTimeWarpDistance: Return the Dynamic Time Warp distance between two temporal values

  • dynTimeWarpPath: Return the correspondence pairs between two temporal values with respect to the Dynamic Time Warp distance

Multidimensional Tiling

Bin Functions

  • valueBins, timeBins: Return a set of bins that cover a value or time span with bins of the same width or duration

  • getValueBin, getTimeBin: Return the bin that contains a number or timestamp

Tile Functions

  • valueTimeTiles, spaceTiles, spaceTimeTiles: Return a set of tiles that cover a temporal or spatiotemporal box with tiles of the same size and duration

  • getValueTimeTile, getSpaceTile, getSpaceTimeTile: Return the tile that contains a value or point and a timestamp

Bounding Box Functions

  • valueSpans, timeSpans: Return an array of value or time spans obtained from the instants or segments of a temporal value with respect to a value or time grid

  • valueTimeBoxes: Return an array of temporal boxes obtained from the instants or segments of a temporal number with respect to a value and time grid

  • spaceBoxes, spaceTimeBoxes: Return an array of spatiotemporal boxes obtained from the instants or segments of a temporal point with respect to a space and possibly a time grid

Split Functions

  • valueSplit: Fragment a temporal number with respect to value bins

  • timeSplit: Fragment a temporal value with respect to time bins

  • valueTimeSplit: Fragment a temporal number with respect to a value and time grid

  • spaceSplit: Fragment a temporal point with respect to a spatial grid

  • spaceTimeSplit: Fragment a temporal point with respect to a spatiotemporal grid

Aggregations

  • tCount: Temporal count

  • extent: Bounding box extent

  • tAnd: Temporal and

  • tOr: Temporal or

  • tMin: Temporal minimum

  • tMax: Temporal maximum

  • tSum: Temporal sum

  • tAvg: Temporal average

  • wMin: Window minimum

  • wMax: Window maximum

  • wCount: Window count

  • wSum: Window sum

  • wAvg: Window average

  • tCentroid: Temporal centroid

Utility Functions

  • mobilitydb_version: Return the version of the MobilityDB extension

  • mobilitydb_full_version: Return the versions of the MobilityDB extension and its dependencies
