Comparisons

Ever and Always Comparisons

The probe operand may be a bare h3 H3 cell identifier or another th3 trajectory; both are supported in either operand position.

  • Return true if a temporal H3 cell is ever equal to the probe at at least one instant

    ever_eq({h3,th3},th3) → boolean

    ever_eq(th3,h3) → boolean

    {h3,th3} ?= th3

    th3 ?= h3

    SELECT th3
  '[831c02fffffffff@2001-01-01, 880326b885fffff@2001-01-05]'
  ?= 880326b885fffff::h3;
-- true

  • Return true if a temporal H3 cell is always equal to the probe across its entire time axis

    always_eq({h3,th3},th3) → boolean

    always_eq(th3,h3) → boolean

    {h3,th3} %= th3

    th3 %= h3

  • Return true if a temporal H3 cell is ever different from the probe

    ever_ne({h3,th3},th3) → boolean

    ever_ne(th3,h3) → boolean

    {h3,th3} ?<> th3

    th3 ?<> h3

  • Return true if a temporal H3 cell is always different from the probe

    always_ne({h3,th3},th3) → boolean

    always_ne(th3,h3) → boolean

    {h3,th3} %<> th3

    th3 %<> h3

Temporal Comparisons

  • Return a temporal boolean giving the per-instant equality (or inequality) between a temporal H3 cell and a probe

    temporal_teq({h3,th3},th3) → tbool

    temporal_teq(th3,h3) → tbool

    temporal_tne({h3,th3},th3) → tbool

    temporal_tne(th3,h3) → tbool

    {h3,th3} #= th3

    th3 #= h3

    {h3,th3} #<> th3

    th3 #<> h3
